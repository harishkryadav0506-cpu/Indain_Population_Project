# India Population Prediction 🇮🇳 (Streamlit Version)

A stunning, interactive web application for visualizing India's population trends and predictions from 1950 to 2100.

![Python](https://img.shields.io/badge/Python-3.9+-blue?style=flat-square&logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=flat-square&logo=streamlit&logoColor=white)
![Plotly](https://img.shields.io/badge/Plotly-3f4f75?style=flat-square&logo=plotly)

## 🌟 Features

- **Live Population Counter** — Real-time estimation updating every second
- **Interactive Charts** — Population trends, growth rates & world comparison built natively with Plotly Express
- **Population Predictor** — Slider to predict population for any year (1950-2100)
- **State-wise Data** — Top 8 most populous Indian states
- **Responsive Layout** — Full width sidebar and native dark mode compatibility

## 🚀 Quick Start

### Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

## 📦 Deployment via Streamlit Community Cloud (Easiest)

1. Push your code to GitHub.
2. Go to [share.streamlit.io](https://share.streamlit.io).
3. Sign in with GitHub.
4. Click **Deploy an app**, and paste your GitHub repository link.
5. In **Main file path**, ensure it says `app.py`.
6. Click **Deploy!**

It takes less than 2 minutes and is completely free!
